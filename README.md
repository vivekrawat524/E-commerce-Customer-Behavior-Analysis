# 🛒 E-commerce Customer Behavior Analysis

## 📌 Project Overview
This project analyzes customer behavior in an e-commerce dataset using Python and Exploratory Data Analysis (EDA).  
The goal is to understand customer patterns, spending behavior, and business insights.

---

## 🎯 Objectives
- Understand customer purchasing behavior  
- Identify high-value customers  
- Analyze city-wise and gender-wise distribution  
- Study spending patterns and satisfaction levels  
- Extract business insights from data  

---

## 📊 Dataset Information
The dataset contains customer details such as:
- Customer ID  
- Gender  
- Age  
- City  
- Membership Type  
- Total Spend  
- Items Purchased  
- Satisfaction Level  
- Discount Applied  
- Days Since Last Purchase  

---

## 🛠️ Tools & Technologies Used
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

---

## 🔍 Key Insights
- A small group of customers contributes most of the revenue  
- Certain cities dominate the customer base  
- Spending behavior varies across age groups  
- Discounts influence purchasing decisions  
- Customer satisfaction is generally positive  
- Customers show different engagement levels  

---

## 📈 Visualizations Used
- Bar Charts  
- Pie Charts  
- Line Charts  
- Histograms  

---

## 🚀 Project Workflow
1. Data Loading  
2. Data Cleaning  
3. Exploratory Data Analysis (EDA)  
4. Insights Generation  
5. Data Visualization  
6. Conclusion  

---

## 📌 Conclusion
This analysis helps businesses understand customer behavior and improve marketing strategies, customer retention, and sales performance.
